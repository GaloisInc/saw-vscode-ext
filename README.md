# SAW VSCode Extension

A [SAW](https://saw.galois.com) (Software Analysis Workbench) syntax highlighting extension for VSCode

## Features

* Syntax highlighting
* Bracket matching
* Bracket autoclosing
* Bracket autosurrounding
* Comment toggling
* Auto indentation
